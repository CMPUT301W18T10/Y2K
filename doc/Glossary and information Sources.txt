\uc0\u9679      UserR = Requester:\
\
A the person who asks for something or who makes a request.  \
A user requester is a user who is asking for something to be done at the lowest amount, in our case, code to be completed.\
\
\
\uc0\u9679      UserP= Provider:\
\
A person that provides something.\
A task for some amount, in our case a user provider would provide code to a code requester for a certain amount of money.\
\
\
\uc0\u9679      User= Requester and/orProvider:\
\
A person who uses or operates something.\
A user using our app can be both a user requester(code requested to be completed) or a user provider wanting to complete code for some amount of money.                                              \
